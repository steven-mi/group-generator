# Group Generator

A web application to split your list of items in random subgroups.

![](.github/img.png)

## Getting Started

Install dependencies

```bash
yarn install
```

**Start application**

```
yarn start
```

**Start tailwind viewer**

```
yarn run tailwind
```

**Start storybook**

```
yarn run storybook
```
