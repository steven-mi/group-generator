# Group Generator

A web application to split your list of items in random subgroups.

**Technology Stack**

- [ReactJS Frontend](https://reactjs.org/)
- [TailWindCSS](https://tailwindcss.com/)

## Getting Started

Install dependencies
```bash
npm install
```

**Start application**

```
npm start
```
