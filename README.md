# Group Generator

A web application to split your list of items in random subgroups.

## Getting Started

Install dependencies

```bash
npm install
```

**Start application**

```
npm start
```

**Start tailwind viewer**

```
npm run tailwind
```

**Start storybook**

```
npm run storybook
```
